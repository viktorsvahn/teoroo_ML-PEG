# Teoroo ML-PEG
Data made accessible for test purposes related to [ML-PEG](https://ml-peg.stfc.ac.uk/).

# How-to
1. Make sure you have the atest version ready by running `pip install mace-torch --upgrade`.
2. Create your calc and analysis scripts and place them in the directories
3. Run the test using `ml_peg calc --category electric_field --test scaling_pol`
